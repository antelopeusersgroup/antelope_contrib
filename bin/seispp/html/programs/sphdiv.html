sphdiv

Usage:

sphdiv [-decay power -text --help] < infile >outfile

Algorithm:

Applies a constant geometric spreading correction to ThreeComponentSeismogram objects grouped into one or more ThreeComponentEnsemble objects stored in the input file, infile.   The algorithm requires each seismogram to set a real-valued header (Metadata) attribute with the tag offset.   Each seismogram is scaled by offset^power where power is a variable that can be changed by the -decay argument.  If the retrieved offset is zero, the program sets offset to a 0.001.   That probably should be a parameter, but if you need that it would be easy code or just edit this code for your own use.

Important to emphasize that this program computes and applies a constant computed gain for each seismogram.  

Options:

-decay power
Set the decay constant to the value of power.  The default is 2.0.

-text
Write the output in text format.  Default is a binary serialized file.

--help
Echo the usage line.
