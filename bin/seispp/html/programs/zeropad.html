zeropad

Usage:

zeropad [-pad dt -taper dt -text --help] < in > out

Algorithm:

Zero pads the front end of a file of TimeSeriesEnsemble objects (file "in" read through stdin) a specified number of sections.   Always applies a linear taper at the front end of the input time series.  To turn the taper off just make the length one sample.  

The program will skip any ensemble where the taper length exceeds half the length of any seismogram in the ensemble.   An error message will be posted to stderr when this happens.

Options:

-pad dt
Change the zero pad length to dt s.  Default is 0.1 s.

-taper dt
Change the linear taper length to dt seconds.  Default is 0.01 s (effectively no taper for most broadband data).

-text
Write the output in text format.  Default is a binary serialized file.

--help
Echo the usage line. 

-v
More verbose output to stderr.
