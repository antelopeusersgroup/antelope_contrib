dismember

Usage:

dismember [-scalar -binary] < infile > outfile

Algorithm:

Break ensembles into member seismograms.  In the seispp library an ensemble is a group of objects bundled and stored in an std::vector container.   In seispp the objects are either scalar or vector (three-component) seismograms.  It is sometimes useful to ungroup the ensembles.  This does that and writes an ungroup set of member objects to output.  

Options:

-scalar - Assume input is TimeSeriesEnsemble objects so output will be a series of TimeSeries objects.  Default is ThreeComponentEnsemble and ThreeComponentSeismograms respectively.
-binary - assume binary format for input and output (default is text format)
