extract_component

Usage:

extract_component n [--help -text] < infile > outfile

Algorithm:

Extract a single component from a file of ThreeComponentEnsemble objects to produce a series of TimeSeriesEnsemble objects with only the requested component.  One argument, n in the usage line above, is required.   n must be 0, 1, or 2 and is used to selected the desired component from the 3C seismograms.   Note the C order so 0 selects the x1 component, 1 the x2 component, and 2 the x3 component.  

Options:

-scalar - Assume input is TimeSeriesEnsemble objects so output will be a series of TimeSeries objects.  Default is ThreeComponentEnsemble and ThreeComponentSeismograms respectively.
-text
Write the output in text format.  Default is a binary serialized file.
