seispp_b2t and seispp_t2b

Usage:

seispp_t2b [--help] < in > out
seispp_b2t [--help] < in > out

Algorithm:

The boost serialization library used by this package of programs supports binary and text output formats.   Both are opaque to a human reader, but a text file is theoretically easy to transport between machines and is the default for all applications.   This pair of filters converts files of ThreeComponentEnsemble objects between the two formats.   seispp_b2t converts binary to text format and seispp_t2b does the opposite.

Note the strong limitation that only serialized ThreeComponentEnsemble object are currently supported. 

Options:

--help - prints the usage line
