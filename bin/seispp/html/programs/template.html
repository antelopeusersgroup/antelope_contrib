This set of options are common and included in this template.  Delete if they aren't supported or edit if necessary.

-t object_type
Change type of objects expected in serialized data stream for input and output.   Default is ThreeComponentEnsemble.   Alternatives are TimeSeries, ThreeComponentSeismogram, and TimeSeriesEnsemble.

-pf pffile
Use pffile instead of default (always the name of the main program)

-text
Write the output in text format.  Default is binary serialized file.

-v
Be more verbose.  Note all log information goes to stderr so it is best to use this option only for debugging problems.

--help
Echo the usage line.
