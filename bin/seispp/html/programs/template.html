Options:

All seispp modules should support these three options - they don't yet, but this defines the pattern.

-binary
Write the output in binary.  Default is text serialized file.

-v
Be more verbose.  Note all log information goes to stderr so it is best to use this option only for debugging problems.

--help
Echo the usage line.
